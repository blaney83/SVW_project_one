Let's Travel!
  • Let's Travel(current)
  • Documentation

Let's Travel's basic functionality is for time management of driving. It will show the user's trip time for regularly visited locations like work and home.


Let's Travel will already know the user's current location. To add a destination; the user must add a their destination with the add button feature. To add a button user must fill out destination name and address. Then click add destination button.


The location will be added to the table below with the information trip time, arrival time, and distance. With the added feature of weather for your current location and destination. Trip time does take traffic, construction, accidents, etc into concideration when calculating.


To remove a saved destination user simply clicks the remove feature. It is the minus on the right of the button.


If the user wishes to open route in google maps. The user can click on the google maps icon, and it will open the route clicked in google maps.
